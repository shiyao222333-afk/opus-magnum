# README 标准模板（巨作 Opus Magnum）

> 本文件是「五器工坊」各项目 README.md 的统一结构标准。
> 每个项目（Nigredo 馏析 / Albedo 炼真 / Citrinitas 熔知 / Rubedo 凝华 / Opus Magnum 巨作）的 README.md
> 都应**照此结构生成**，保证四个说明文件风格一致、信息齐全、便于横向对比。
>
> 使用方式：复制本文件为各项目根目录的 `README.md`，把 `{{占位符}}` 替换为真实内容，删掉「写作提示」注释。

---

## 写作总体规则

1. **纯中文**，不写英文段落（标题可保留英文项目原名作对照）。
2. **说人话**：每节第一句讲清"这是什么、解决什么问题"，避免一上来就堆技术名词。
3. **竞品对比只用一张表**（见下方「核心能力 & 竞品对比」节），把"各有千秋"收进该表最后一行，不得另起第二张表。
4. **版本号**以项目代码内 `__version__` 为准；路线图里"已完成"打 ✅、"规划中"打 🔮。
5. **技术栈**必须标注每个关键依赖的授权协议（MIT / Apache / BSD 安全；AGPL/LGPL 禁止用于闭源核心）。

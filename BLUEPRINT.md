## 🎯 愿景

**一人即公司，个体即超级。**

一个普通人，借助 AI 工具链，完成从「学习模仿 → 真伪验证 → 知识积累 → 内容创作 → 多平台分发 → 私域经营」的完整闭环。

如同炼金术士将铅转化为黄金——我们通过模仿成功者的方法，借助 AI 加速执行，将互联网上的噪音转化为可盈利的知识资产。

---

## 🧭 核心原则

1. **模仿优先于创造** — 别自己想怎么做，找已经做成功的人，拆解他们的方法，复制、改良、超越
2. **MVP 优先** — 做最少的事验证最重要的问题，不做提前优化
3. **自动化优先** — 能自动不要手动，零人工干预，全自动管道
4. **程序优先于大模型** — 能写死规则不用 AI，省钱也省时间
5. **质量第一** — 宁可慢，不将就；知识库不能被虚假信息污染

---

## 📡 战略雷达

OpusMagnum 不仅是总蓝图，还承担**战略情报**职能：

- **每日 AI 新闻扫描** — 自动化任务抓取最新 AI 资讯 + 副业机会
- **四层过滤** — 先匹配现有项目（Citrinitas/Nigredo/Albedo/Rubedo），项目相关的送入对应研究队列
- **超级收纳** — 不匹配任何现有项目、但与一人公司/赚钱/副业相关的情报归入 OpusMagnum 战略储备
- **研究队列** — 所有情报写入 `research-queue.md`，按优先级（🔴🟡⚪💡）分级，手动勾选研究完成

这样保证：现有项目能吃到前沿技术红利，同时不会错漏可能孵化新业务方向的市场信号。

## 🧠 核心战略：模仿成功者

一人公司副业最有效的方法不是自己想一个新路子，而是**找到已经做成功的人，拆解他们的方法，模仿、改良、超越**。

```
找到成功者 → 拆解方法 → 验证真伪 → 存入知识库 → 模仿执行 → 数据反馈 → 改良迭代
     ↑                                                                    │
     └──────────────────── 新成功者出现，继续拆解 ←──────────────────────────┘
```

### 四器分工

| 器 | 在这个战略中的角色 |
|----|-----------------|
| **Nigredo 馏析** | 拆解成功者——下载他们的视频、提取文案、结构化分析 |
| **Albedo 炼真** | 验证真假——成功者说的一定对吗？跨源比对，拦截虚假信息 |

| **Rubedo 凝华** | 执行赚钱——把知识库里学来的方法，用 SOP 自动化跑起来 |

### 🏭 Citrinitas · 熔知 — 核心枢纽 · 知识引擎

> *黄化是觉醒的时刻。熔炉之火不灭，万物经此锻造成型。*

| 属性 | 值 |
|------|-----|
| **功能** | 知识库的存储、检索、管理——**整个工坊的唯一数据中心** |
| **输入** | Nigredo 产出的结构化笔记、Albedo 验证后的可信知识、手动注入的参考书目 |
| **核心** | 混合检索（BM25稀疏+密集向量）+ 重排序 + 置信度路由审核队列 |
| **版本** | ✅ v0.8.0（搜索优化 + 审核队列全线完成） |

**地位**: 整个系统的中央知识库。所有其他工具产出的知识最终流入 Citrinitas。所有其他工具也从中检索知识来辅助决策。Citrinitas 先稳定，其他工具才有意义。

### ⚗️ Nigredo · 馏析 — 学习模仿引擎

> *黑化是第一道工序。万物先分解，回归混沌，方能重生。*

| 属性 | 值 |
|------|-----|
| **功能** | 下载成功者的内容 → 语音转文字 → 结构化分析 → 喂入 Citrinitas |
| **输入** | B站一人公司/副业类视频 → 未来: YouTube, 小红书, 播客 |
| **核心** | 视频下载 + 语音转文字 + AI 结构化（商业模式拆解/话术分析/情绪曲线） |
| **状态** | 📋 规划完成，赚钱稳定后启动 |

**地位**: 外部信息的主要入口。核心逻辑——副业最快的方式就是模仿成功者。Nigredo 负责把"别人的成功"拆解成"你能学的东西"。

**三种拆解模式**:
| 模式 | 输入 | 输出 |
|------|------|------|
| 📖 案例学习 | "一人公司""副业收入"类视频 | 商业模式、收入结构、关键决策点 |
| ✍️ 文案模仿 | 小红书/公众号爆款 | 结构拆解、话术模板、情绪曲线 |
| 🔥 竞品分析 | 同类UP主/博主 | 横向对比、内容策略、差异化角度 |

### 🔬 Albedo · 炼真 — 知识净化器

> *白化是净化。洗涤污秽，去除杂质，唯有真金留存。*

| 属性 | 值 |
|------|-----|
| **功能** | 跨源信息比对 + 矛盾检测 + 可信度评分——**在知识进入 Citrinitas 前把关** |
| **输入** | Citrinitas 文档 + Nigredo 产出的笔记 + 任意文本源 |
| **核心** | 声明提取 → NLI矛盾检测 → 证据融合 → 可信度评分 |
| **状态** | 📋 深度规划完成，待开发 |

**地位**: 模仿成功者的路上，最大的坑不是学不会，而是学到了假的东西。Albedo 就是这条路上的防火墙。

| 场景 | 没 Albedo 会怎样 | 有 Albedo 会怎样 |
|------|----------------|-----------------|
| 两UP主说相反的话 | 不知道信谁，凭感觉 | 自动标注矛盾，附可信度评分 |
| "大佬"分享的方法 | 直接模仿，可能跑偏 | 跨源验证，不实信息入库前被拦截 |
| 知识库长期积累 | 假信息悄悄混入，越学越偏 | 每张知识卡标注可信度，源头可追溯 |

### ✨ Rubedo · 凝华 — 副业 SOP 自动化平台

> *红化是最后的蜕变。哲人石诞生，一滴即可点石成金。*

| 属性 | 值 |
|------|-----|
| **功能** | AI深度介入的副业SOP自动化平台：共用工具层 + SOP独特工具层 |
| **共用工具层** | 时间记录 / AI对话工具箱 / 数据分析 / 任务调度（任何SOP都能用） |
| **SOP工具层** | 酷家乐工具包 / 小红书工具包 / 未来其他SOP工具包 |
| **核心** | 从 Citrinitas 检索模仿素材 → AI 辅助执行 → 数据回流优化 |
| **版本** | 🚧 v0.1（时间记录工具已上线，仓库已创建） |

**地位**: 变现的"最后一公里"。不只是发布内容——而是把 Citrinitas 里积累的"别人怎么做"变成自动化 SOP，让你能用更少的时间做更多的事。

**工具平台架构**（详细见 `docs/rubedo-architecture.md`）：
- **共用工具层**：时间记录工具、AI对话工具箱（需求确认/报价/合同）、数据分析工具、自动化任务调度
- **SOP独特工具层**：每个副业项目一套工具包（酷家乐SOP工具包、小红书SOP工具包...）

## 🛡️ 护城河

### 1. 模仿飞轮
每器不是孤立的——它们互相喂养：
- Nigredo 拆解成功者 → Albedo 验证 → Citrinitas 存储
- Citrinitas 的知识 → 供给 Rubedo 自动化执行
- Rubedo 的执行数据 → 回流 Citrinitas（什么方法有效？时薪多少？）
- 新的成功者出现 → Nigredo 继续拆解 → 飞轮加速

### 2. 个人知识图谱
系统积累的不是通用数据，而是**你**的阅读习惯、兴趣领域、验证标准。你模仿过的每一个成功者的方法，都变成了可检索、可复用的知识资产。

### 3. 跨源矛盾检测
目前市面上没有任何产品能做到「跨 B站视频 + 个人知识库 + 网页文章」的统一矛盾检测。Albedo 是核心壁垒。

### 4. 端到端闭环
从找到成功者到模仿赚钱是完整的，不是拼凑的工具链。

### 5. 开源但不可复制
代码开源供参考，但具体配置、知识库、调优参数是私有的。架构理解不难，积累到同等量级需要时间。

### 6. 品牌壁垒
炼金术命名体系天然具有辨识度和故事性，五器工坊的概念可以注册商标。
